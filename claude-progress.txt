# Hi-Tom-AI 项目进度

## 项目概述
AI 电商内容生成平台重构项目

### 目标
1. Provider 配置化 - 模型、API 参数可在管理后台配置
2. 计费配置化 - 每个模型的积分消耗规则可配置
3. Vercel Functions 代理 - AI 请求通过 Serverless 转发
4. 前端保留 - Vue 前端 UI 基本不变
5. 后端精简 - 保留业务逻辑，AI 代理迁移到 Vercel

### 架构
```
用户浏览器
    │
    ├── AI 请求 → Vercel Functions → AI 服务商
    │
    └── 业务请求 → 你的服务器 (FastAPI + SQLite)
```
